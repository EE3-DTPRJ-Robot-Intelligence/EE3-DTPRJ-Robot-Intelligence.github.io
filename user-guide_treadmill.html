User Guide: Treadmill

Controlling treadmill speed directly using Arduino UNO.

GitHub Repository

Prerequisites

Hardware

  • 1 x Arduino UNO
  • 1 x treadmill (model: Dream Treadmill D1000): a signal cable with a 10-pin connector from the treadmill is to be used for direct signal control.
  • 1 x treadmill-Arduino interface board: a stripboard with a 10-pin header.

Software

  • Arduino IDE

Setup

Hardware

Treadmill signal cable colour coding scheme:
Name Colour Description
Vrefblack5V
VRwhite~4.73V
GNDgrey0V (ground)
RPMpurpleRPM signal (active low, 2 peaks per rotation)
VDDblue~15V
PWMgreenPWM signal (native controller use 400Hz)
DNyellowinclination up (active high)
UPorangeinclination down (active high)
S/Wred~15V
relaybrownrelay signal (active high)
  1. Connect treadmill signal cable to the interface board.
  2. There are 4 coloured wires on interface board that need to connected to Arduino UNO:
    • Green (PWM): to pin 2
    • Purple (RPM): to pin 9
    • Brown (relay): to pin 5
    • Gray (GND): to any ground pin (GND)
    As the VDD and S/W pins on the signal cable carry a voltage of 15V, do not connect them to any of the Arduino pins, otherwise the Arduino UNO will be damaged.

Usage

  1. Upload the program treadmill-dt-d1000.ino to Arduino UNO by using Arduino IDE.
  2. Power on the treadmill.
  3. Open the Serial Monitor in Arduino IDE.
  4. Sending commands:
    • To turn on or off the motor: send on or off via Serial Monitor
    • To set new PWM value: send p<pwm> via Serial, where <pwm> is an integer in the range of 0 to 65535 (default PWM on start-up is 0)

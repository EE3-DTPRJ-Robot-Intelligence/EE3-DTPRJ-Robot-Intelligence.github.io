User Guide: Treadmill

Controlling treadmill speed directly using Arduino UNO

Prerequisites

Hardware

  • Treadmill, model: Dream Treadmill D1000
  • Arduino UNO
  • Treadmill-Arduino interface board: a stripboard with a 10-pin header.

Software

  • Arduino IDE

Usage

  • Upload code to Arduino UNO by using Arduino IDE.
  • Connect treadmill 10-pin cable to the interface board.
  • Open the Serial Monitor in Arduino IDE.
  • To turn on or off the motor: send on or off via Serial Monitor
  • To set new PWM value: send p<pwm> via Serial, where <pwm> is an integer in the range of 0 to 65535 (default PWM on start-up is 0)
